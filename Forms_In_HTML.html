Forms in HTML

Forms are used to collect information/input from user. This information is sent to server for the processing.

Form tag is used to create HTML form for user input The form element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc

HTML input element

An input element can be displayed in many ways, depending on the type attribute it takes

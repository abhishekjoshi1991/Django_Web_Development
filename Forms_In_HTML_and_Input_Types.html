Forms in HTML

Forms are used to collect information/input from user. This information is sent to server for the processing.

Form tag is used to create HTML form for user input The form element is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc

HTML input element

An input element can be displayed in many ways, depending on the type attribute it takes

  2. Input Type Password

  3. Input Type Submit

    4. defines a button for submitting form data to a form-handler.

  4. Input Type Reset

    5. reset button will reset all form values to their default values (if provided) or blank value



  5. Input Type Radio

    6. Radio buttons let a user select ONLY ONE choice out of number of choices




  6. Input Type Checkbox

    7. let a user select ZERO or MORE options of a limited number of choices




  7. Input Type Button

  8. Input Type Color

    9. Depending on browser support, a color picker can show up in the input field.

  9. Input Type Date

    10. Depending on browser support, a date picker can show up in the input field.

  10. Input Type Datetime-local

    11. This specifies a date and time input field, with no time zone.

  11. Input Type Email

    12. This is used for input fields that should contain an e-mail address.Depending on browser support, the e-mail address can be automatically validated when submitted..

  12. Input Type File

    13. This defines a file-select field and a "Browse" button for file uploads.

  13. Input Type Hidden

    14. This defines hidden input field (not visible to user while filling the form.) It allows to include data that cannot be seen or modified by users when a form is submitted

  14. Input Type Month

    15. This allows user to select month and year.

  15. Input Type Week

    16. This allows user to select week and year.

  16. Input Type time

    17. This allows user to select time.(not time zone specific)

  17. Input Type Number:

    18. This allows user to define a numeric field. We can also set min and max restriction for number to be selected.

    Follwong is example to select number from 1 to 100, with step of 10 and default value of 30.

  18. Input Type Range

    19. Defines a control for entering the number as slider control. Default range is 1 to 100. We can change the range using min max attribute.

Anchor Tag

The anchor tag defines a hyperlink, which is used to navigate from one page to another.

The most important attribute is the href attribute, which indicates the link's destination

Image tag

The tag is used to embed an image in an HTML page.

The tag has two required attributes:

  1. src - Specifies the path to the image
  2. alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed

To link an image to another document, simply nest the image tag inside an anchor tag
